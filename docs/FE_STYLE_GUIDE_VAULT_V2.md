# Vault v2.0 FE 스타일 가이드 (보스 요구 반영)

## 1. 메타
- 문서 타입: FE 스타일 가이드
- 버전: v0.1 (초안)
- 작성일: 2025-12-20
- 대상: 프론트엔드/디자이너/기획자

## 2. 비주얼 톤
- 배경: #050505 라디얼 그라디언트(센터 밝음 → 가장자리 암흑), 고정 배경 스크롤
- 포인트 컬러: GOLD #D2FD9C, PLATINUM #07AF4D, DIAMOND #0AA787, 경고/시한 #F97935
- 카드: 20px+ 라운드, 헤더 그라디언트 + 어두운 바디, 1~2px 하이라이트 보더, 블러/글로우 그림자
- 배지: 상태/리워드 배지 모두 그라디언트 + drop-shadow; AVAILABLE는 우상단 대각 배지
- 폰트: Noto Sans KR 400/500/700, 숫자/타이머는 모노스페이스(예: 'Roboto Mono' 대체)로 ms 표기 시 가독성 확보

## 3. 레이아웃
- 헤더: 그라디언트 타이틀 + NEW 라벨, 얇은 글로우 언더라인
- 서브배지: "이벤트 종료까지 N일 N시간" + LIMITED EVENT 토큰 (반투명 블랙 캡슐)
- 카드 그리드: 모바일 1열 스택, 태블릿 2열, 데스크톱 3열; 카드 클릭 시 강조 보더/글로우
- 플로팅 손실 배너: 모바일 하단 스티키, 데스크톱 우하단; ms 타이머 연동, CTA "지금 회수"
- 완성 보너스: 별/오브젝트 회전 + 그라디언트 프로그레스 바, 3단계 마커

## 4. 컴포넌트 토큰
- 버튼
  - Enabled: 그라디언트(bg from tier.from to tier.to), 글로우 그림자, hover시 색상 시프트
  - Disabled: desat + 반투명 보더, 커서 default
  - Shine: 2s 주기 슬라이드 그라디언트
- 상태 배지
  - LOCKED: 주황 그라디언트, 자물쇠 아이콘
  - AVAILABLE: 초록 그라디언트, 눈 아이콘
  - COMPLETED: 회색/실버 그라디언트, 체크 아이콘
- 체크리스트
  - 배경: 블랙/20 투명 + 보더
  - 체크 원: 그라디언트 필 + 글로우; 미완은 회색 원
  - 힌트 텍스트: 이탤릭, 회색
- 진행률 바
  - 다이아: 네온 그라디언트 바 + 끝 하이라이트
  - 보너스: 오렌지→그린 그라디언트, 3개 마커
- 타이머
  - 기본: 주황 배경 캡슐, 초 단위
  - 긴급(<1h): ms 표기, 저채도 점멸, 모노스페이스 숫자, aria-live polite

## 5. 모션
- 카드 hover: y -5px, 보더/그림자 컬러 시프트
- 아이콘: 부유 애니메이션(y ±5px) + 글로우 펄스
- 배지/토스트: scale-in + fade
- 버튼: hover scale 1.02, tap 0.98
- 프로그레스/shine: 2s~3s 주기 반복, prefers-reduced-motion 시 중단

## 6. 접근성/국제화
- 대비: 주요 텍스트 대비 4.5:1 이상 유지, 네온 색상 위에 그림자/아웃라인 추가
- aria-label: 금액/타이머/CTA 모두 라벨 제공
- 모션 토글: prefers-reduced-motion에 따라 애니메이션 비활성
- 다국어: 텍스트는 i18n 키로, 금액/타이머 포맷은 로캘별 변환

## 7. 구현 메모
- 전역 폰트 로드: Noto Sans KR + 모노스페이스 서브셋
- 그라디언트/색상은 CSS 변수화 (--vault-gold, --vault-plat, --vault-dia, --accent-warn)
- 반투명/블러는 backdrop-filter 사용, Fallback 필요 시 불투명 대안 제공
- 카드/배지/토스트에 drop-shadow(filter) 대신 box-shadow와 혼합 사용해 퍼포먼스 균형

## 8. 참고 스냅샷
- 제공된 보스 디자인 스크린샷을 기준으로 컬러/배치/텍스처를 최대한 준수
