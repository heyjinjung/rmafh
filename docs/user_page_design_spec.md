# 유저 페이지 디자인 명세서 (v2.1)

## 1. 개요
본 문서는 CC 카지노 금고 유저 페이지(`frontend/pages/index.jsx`)의 개선된 디자인 명세(v2.1)를 기술합니다. **가독성(Contrast Ratio 4.5:1 준수)**과 **심플함**을 강화하고, 사이드바를 간소화하는 것이 핵심입니다.

## 2. 주요 변경 사항 (v2.1)
- **배경**: 깊이감 있는 **단색(Flat) 다크 그린** 배경으로 변경 (텍스처 제거).
- **가독성**: 다이아몬드 등급 컬러를 **Bright Cyan (#00E0FF)**으로 변경하여 어두운 배경에서의 대비 확보.
- **사이드바**: 장식적 요소(테두리, 글로우)를 제거하고 **정보 전달 위주**로 구조 단순화.
- **프로필**: 하단 프로필 카드 UI 제거 → 상단 헤더 직후 닉네임/로그아웃 단순 배치.

## 3. 디자인 토큰

| 토큰명 | Hex 값 | 용도 |
| :--- | :--- | :--- |
| **Primary Accent** | `#D2FD9C` | 메인 강조, 텍스트 포인트 (네온 그린) |
| **Secondary Accent** | `#07AF4D` | 플래티넘 등급 아이덴티티 (딥 그린) |
| **Tertiary Accent** | `#00E0FF` | **[변경]** 다이아몬드 등급 (브라이트 시안) - 고대비 |
| **Background Dark** | `#051a10` | 버튼 배경, 카드 배경 (매우 어두운 녹색) |

| **Text Sub** | `#889988` | 보조 텍스트 |

### 타이포그래피 (Typography)
- **제목**: *Noto Sans KR* (Bold)
- **본문**: *IBM Plex Sans* (Regular)

## 4. UI 컴포넌트 명세

### A. 사이드바 네비게이션
- **스타일**: 다크 프로스티드 글래스 (`backdrop-filter: blur(20px)`).
    -   **[제거]** 우측 테두리 (Border-right).
    -   **[제거]** 컨테이너 호버 글로우 효과.
- **구조 (상단 → 하단 순서)**:
    1.  **Nav**: 로고 (CC CASINO).
    2.  **헤더**: 타이틀 ("신규회원 전용금고").
    3.  **유저 정보**: 닉네임 + 로그아웃 버튼 (심플한 텍스트/버튼 형태).
    4.  **CTA 버튼**: 2개의 정사각 버튼 (카지노 이동 / 공지 채널).
    5.  **하단 링크**: 금고 가이드북 (텍스트 링크).

### B. 메인 콘텐츠 (금고 카드)
- **다이아몬드 카드**: 테두리와 아이콘 글로우 색상을 `#00E0FF`로 변경하여 시인성 극대화.
- **배경**: 그라디언트나 텍스처가 없는 차분한 `#020b07` 컬러 적용.
